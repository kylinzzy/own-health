# 健康瘦身 - 个人健康管理小程序

一款纯私人自用的健康减肥健身微信小程序，无商业化、无多用户体系，主打个人数据本地留存。

## 功能模块

### 1. 居民膳食指南专区
- 收录《中国居民膳食指南》权威核心内容
- 按膳食宝塔、营养素推荐、不同人群饮食原则分类展示
- 支持关键词搜索

### 2. 减脂食谱库
- 分类收录热门减脂食谱
- 支持按餐次、热量区间、食材类型筛选
- 支持收藏和一键添加到饮食记录

### 3. 体重管理系统
- 每日体重记录（支持体脂率）
- 自动生成体重趋势曲线图
- 每周自动生成体重总结

### 4. 饮食记录功能
- 按早餐/午餐/晚餐/加餐分类记录
- 支持食谱选择和手动输入
- 自动汇总每日营养数据

### 5. 运动健身记录
- 记录运动项目、时长、消耗热量
- 自动计算每日热量缺口
- 周运动统计

### 6. 智能健康分析
- 基于饮食/运动/体重数据生成评估报告
- 多维度健康评分
- 针对性调整建议

## 技术特点

- **纯本地存储**：所有数据保存在手机本地，无需后端服务
- **轻量图表**：自研 Canvas 图表，无第三方依赖
- **简约设计**：界面清爽，操作轻量化
- **模块化代码**：核心逻辑附带清晰注释

## 快速开始

1. 注册微信小程序账号
2. 下载微信开发者工具
3. 导入项目，修改 `project.config.json` 中的 `appid`
4. 编译预览，提交审核

详细部署说明见 `docs/DEPLOY.md`

## 项目结构

```
health-miniapp/
├── app.js              # 应用入口
├── app.json            # 全局配置
├── app.wxss            # 全局样式
├── pages/              # 页面目录
├── utils/              # 工具函数
├── data/               # 静态数据
└── docs/               # 文档
```

## 技术栈

- 微信原生小程序框架
- WXSS + CSS 变量
- Canvas API 图表
- wx.storage 本地存储

## 隐私说明

- 所有数据仅存储在用户手机本地
- 不上传任何个人隐私数据到服务器
- 无需网络请求权限
